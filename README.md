# Site_Clone
FOR LEGITIMATE USE CASE ONLY!!! This is use to clone any static website on the net. The full design, assets and all.

FRAMEWORK
This script is built using python. make sure you have python installed.

Usage:
    python3 clone=site.py <target_url> <output_dir>

Example:
    python3 clone=site.py https://www.website.com/ website-clone
    python3 clone=site.py https://www.anotherwebsite.com anotherwebsite-clone

External Dependencies: None (uses only Python standard library)
